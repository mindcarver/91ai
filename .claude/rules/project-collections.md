# 项目收藏文档规则

适用于 `docs/project-collections/` 下的分类收藏文件。

## 格式结构

每个文件遵循统一格式：

1. `# 分类标题` — h1 标题
2. 一段简短描述（1-3 句话）
3. `数量：N` — 单独一行标注项目数量
4. 空行
5. Markdown 表格，列格式：`| 名称 | 概括 | 标签 | 链接 |`

## 格式要求

- 不需要固定的 h2 章节结构，内容以表格为主
- 表格中每行是一个项目条目
- 链接列使用 `[显示文字](URL)` 格式
- 标签用简短关键词
- 新增项目后更新文件头部的数量行
- `README.md` 维护重点板块索引和总计数
