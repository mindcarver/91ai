# AI Coding 工具文档规则

适用于 `docs/ai-coding/` 下的工具评测文件（如 `claude-code.md`、`cursor.md`）。

## 必须包含的章节

每个工具文件必须包含以下 h2 章节（顺序可调整）：

- `## 推荐阅读顺序`
- `## 官方资料`
- `## 学习资料`
- `## 生态项目与能带来的东西`
- `## 使用场景分类`
- `## 评测维度建议` 或等效的评测章节
- `## 当前判断`

## 格式要求

- 文件开头：`# 工具名` + 空行 + `> 更新日期：YYYY/MM 或简要描述`
- 列表统一使用 `-`，不用 `*`
- 外部链接用 `[显示文字](URL)` 格式
- 避免营销话术，说明适合谁、提供什么、有什么限制
