## 如何使用

- 想找工具：先从重点板块进入，再到细分目录定位。
- 想按场景找：先看 [按场景找项目](./by-scenario.md)，再到具体分类。
- 想按自己的身份学习：先看 [按身份推荐](./recommended-by-role.md)，再进入具体分类。
- 想做专题：从对应分类里挑选项目继续深入整理。
- 想做评测：先选定同类项目，再定义统一任务和评测维度。
- 想补充项目：新增到最贴近的分类，并写清楚项目解决什么问题。
