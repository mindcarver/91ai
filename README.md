## 原则

- 资料要全，但不只追求数量。
- 评测要明确，不写模糊的万能推荐。
- 优先真实使用体验，谨慎引用营销话术。
- 关注工程落地、长期维护、安全边界和成本。
- 过时、低质量或失效资料会被移除或标记。
